Marco Points (mPT)

How to install

First of all you need to have metamask installed (It's just a browser extension)


Now you have to add the marcoPoints token in metamask (configure your metamask extension token)

If you want to do it automatically click the below button


If you want to install it manually connect to the deveth network and add the ERC20 token:


  • Contract address

  • Symbol

  • Decimals
